# NBA Dashboard (Streamlit + nba_api)

A simple NBA dashboard built with Streamlit and the `nba_api` library.

## Features
- Standings dashboard (East/West tabs, filters, search)
- Team page (roster + last N games + points chart)
- Player explorer (search player, game log, averages, charts)

## Requirements
- Python >= 3.10 and < 3.13 (recommended: 3.12)
- Poetry

## Setup (Poetry)
```bash
poetry install
